To Run this on Local machine

Paste the folder "Diabetes Prediction" in " C:\Users\your_user  " directory. { Directory present other than the public in "users" folder}

Open Anaconda Prompt

Command : cd "directory of folder" eg: C:\Users\vssal
Command : Predictor.py

Copy the address HTTP address onto any browser and run 
Eg: http://127.0.0.1:5000/

Put the Values
Click on Predict
Prediction is ready
